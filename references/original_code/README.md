# Original Code

参照元コードのスナップショットを保存するディレクトリです。

## 目的

- リファクタリング時の参照として元コードを保持
- ライセンス・出典情報の明確化
- 元の実装との比較

## ディレクトリ構成

```
original_code/
└── [source_name]/
    ├── README.md      # 出典・ライセンス情報
    ├── LICENSE        # 元コードのライセンス
    └── original/      # 元コードのスナップショット
```

## README.md テンプレート

```markdown
# [Source Name]

## 出典
- Repository: [URL]
- Commit/Tag: [バージョン情報]
- 取得日: YYYY-MM-DD

## ライセンス
[元コードのライセンスを記載]

## 説明
[どのアルゴリズムに対応するか、何を参照したか]

## 変更点
- algorithms/xxx/ でリファクタリング済み
- 主な変更: [変更内容]
```

## 注意事項

- ライセンスが許可する範囲で使用すること
- 元コードの著作権表示を保持すること
- 大きなファイルは除外し、必要最小限を保存
